## Compiling

1. Install `qt5` (`sudo apt-get install qt5-default`)
2. Run `qmake -v` and check that the Qt version is 5.
3. Run the following commands:
```
cd resident-seal/src
qmake src.pro
make
```
4. Run the executable in `bin`:
```
 ../bin/resident-seal
```
